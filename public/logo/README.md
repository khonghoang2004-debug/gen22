# Logo Files

Place your logo file in this directory.

## Supported File

### Required:
- `gen22-logo2.png` - Main logo file (PNG format)

The system will automatically use this file. If the file is not found or fails to load, text "GEN22 BarberShop" will be displayed as fallback.

## Usage

Simply place `gen22-logo2.png` in this folder:
- The system will automatically detect and use it
- If the file is not found, text "GEN22 BarberShop" will be displayed
- No code changes needed!

## File Naming

**Supported filenames:**
- `logo.png` or `gen22-logo.png`
- `logo.svg` or `gen22-logo.svg` (preferred)

**Important:** Use exact filenames (case-sensitive):
- ✅ `logo.png` or `gen22-logo.png`
- ❌ `Logo.png` or `LOGO.png` (won't work)

## Recommended Sizes

- **SVG**: Any size (scalable)
- **PNG**: 
  - Minimum: 200px width
  - Recommended: 400px width
  - Format: PNG with transparency

## How to Replace Logo

1. Delete old logo files (if any)
2. Drop new logo file(s) into this folder
3. Rename to match exact filenames above
4. Refresh the website - logo will update automatically!

No code changes needed! 🎉
